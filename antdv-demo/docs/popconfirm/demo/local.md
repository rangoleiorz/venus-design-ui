#### 国际化
使用 `okText` 和 `cancelText` 自定义按钮文字。
</cn>

<us>
#### Locale text
Set `okText` and `cancelText` props to customize the button's labels.
</us>

```vue
<template>
  <a-popconfirm title="Are you sure？" ok-text="Yes" cancel-text="No">
    <a href="#">Delete</a>
  </a-popconfirm>
</template>
```
