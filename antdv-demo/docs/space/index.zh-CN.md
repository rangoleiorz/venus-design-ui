## API

| 参数      | 说明     | 类型                                       | 默认值       | 版本  |
| --------- | -------- | ------------------------------------------ | ------------ | ----- |
| align     | 对齐方式 | `start` \| `end` \|`center` \|`baseline`   | -            | 1.6.5 |
| direction | 间距方向 | `vertical` \| `horizontal`                 | `horizontal` | 1.6.5 |
| size      | 间距大小 | `small` \| `middle` \| `large` \| `number` | `small`      | 1.6.5 |
